Basic Tags

html - Creates the HTML document

head - Conains the title, stylesheet and other info

title - Sets the title of the document in the browser's tab; this belongs in the <head> tag

body - Conains all visible content in the document

Formatting

p - Represents a paragraph

br - Breaks a line

blockquote - Represents a block quote

div - Used to formal block content in the stylesheet

span - Used to formal inline content in the stylesheet

Lists

ul - Unordered list

ol - Ordered list

li - Represents an item in the list

Text

h1 -> h6 - Represents a headline, h1 is the largest, h6 is the smallest

b - Represents bold text

cite - Represents a citation
