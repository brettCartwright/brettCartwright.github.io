Selectors

* - all elements

div - all div tags

div,p - all divs and paragraphs

div p - all decendants of divs that are paragraphs

div > p - all direct children of divs that are paragraphs

#name - first element with id "name"

.className - all elements with class "className"

div.className - all divs with class "className"

Psuedo Classes

div:empty - all divs without children

div:first-child - all divs that are the first child of their parent

div:last-child - all divs that are the last child of their parent

div:only-child - all divs that are the only child of thier parent

div:nth-child(2) - all divs whos positions as children of thier parent is divisible by 2
